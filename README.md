# Beverage Sales Dashboard – Power BI  

Dashboard Preview https://ibb.co/v2rTkLW
https://ibb.co/5gjpLvxB

Interactive sales dashboard analyzing *3 million cleaned transactions* (originally 8.9M rows) for a beverage company (2021–2023).

*Live Demo (video)* → [Watch on LinkedIn](http://www.linkedin.com/in/oladotun-olawale)

### Key Metrics
- €391.55M Total Sales (+52.1% YoY)  
- 3M Orders | 10K Customers  
- B2B vs B2C split | Top products | Regional performance  
- Deep customer insights (orders/customer, AOV, discount rate %)

 Features
- 100% custom DAX measures (no quick measures)
- Hand-built Calendar table with full time intelligence
- Dark-mode + neon glow design
- 2 responsive pages: Executive Summary + Customer Insights
- Loads in <3 seconds on 3M rows

 Tech Stack
- Power BI Desktop (November 2025 version)
- DAX, Power Query M, Data Modeling
- Star schema (Calendar ↔ Fact table)

 How to Open

3. Refresh → you’ll see live numbers
